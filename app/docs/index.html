Typography

All of Origin's typography is completely modular. This means you can call the html element as usual:

<foo></foo>

Or you can use the element name as a class name. This will come in handy if you for example want to match the font styling of a heading but still want your text to be displayed inline:

<bar class="foo"></bar>

Headings

All HTML headings, <h1> through <h6>, are available, including the .h1 through .h6 classes. An extra class .h1--boom is added for extra large titles.

The base body copy has been optimised for legibility on both large and small screens. Simply call a <p> element or include a .p class.

Rhythm

Last but not least, the beauty of Origin—the vertical rhythm. All elements of Origin are aligned to a baseline grid defined by the size of the font in <html> and have zero margins by default, everything is a carefully calculated mix of line-height and padding-top, fully responsive to the changes of the font size.

Should you wish to add a little bit of white space in your designs, simply use a .rhythm class on a container. The container itself and all its children will get a calculated vertical margin.
